;climate setup for strat map generation
;climate default must be first
;climate default has height attributes for each zone - the other climates can not have heights set
;the height attributes (two) are the lowest height that band may apply and the max height to which the previous bands may apply.
;the default sea will cascade to other climates sea.
;the default land will cascade to other climates land
;any amendments to a climates land or higher zone will cascade upwards within the climate.
;climate default can not have a color ramp - all other climates can as an option
;you can use tabs or spaces between entries - if you have a space within a path it must be enclosed in " "
;if you want to use a bump for a texture it must be added to EVERY ground_type line you want the bump to apply to
